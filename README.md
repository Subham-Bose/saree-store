# Saree Store
A full-stack saree e-commerce web app built with React (frontend) and Node/Express (backend).
Includes product listing, filters, cart, and API backend.

# Run locally
* git clone https://github.com/Subham-Bose/saree-store.git
* npm install
* npm run dev

# Tech Stack
* React + TypeScript (client)
* Node + Express (server)
* Tailwind CSS
